## Description

This p5.js project demonstrates a game called reflective camera-style experience where a player can navigate through a heavenly
scene/world that is larger than the viewport. This project uses "lerp" to smooth camera movements and emphasize the slow
relaxing theme of the expereince

---

## Learning Goals

- Understand how sound adds to a game experience
- Add a debug menu that allows players to control game state/settings
- Communicate the importance of accessible and simple directions for players
